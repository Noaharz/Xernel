# Architecture Decision Records

Kurze, datierte Notizen zu jeder signifikanten Architektur-Entscheidung.
Format: `000X-kurz-beschreibender-name.md`.

Eine Entscheidung gehört hierhin, sobald sie

- mehrere Subsysteme betrifft, **oder**
- absehbar später hinterfragt wird, **oder**
- nicht aus dem Code allein nachvollziehbar ist.

Bestehende ADRs:

- [0001 — Architektur-Grundentscheidungen für Xernel](0001-architecture-decisions.md)
